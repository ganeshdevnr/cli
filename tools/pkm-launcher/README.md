# PKM (Personal Knowledge Management) Launcher

Quick launcher for Obsidian vaults.

## Features

- Opens Obsidian vaults with a single command
- Supports multiple vaults

## Usage

```bash
# Open personal vault
bin\pkm.bat personal

# Open office vault
bin\pkm.bat office
```

## Requirements

- Obsidian must be installed
- Vaults must be named:
  - `Personal_Vault`
  - `Office_Vault`

## How it works

Uses Obsidian URI protocol to open vaults directly.
